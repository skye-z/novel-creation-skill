# Novel Creation Skill

这是一个为 Trae SOLO Web 优化的中文长篇小说创作 Skill, 适用于内置联网搜索能力、内容提取能力和上下文自我管理能力的智能体

本 SKILL 结合 LinuxDO 论坛各大佬友的分享以及 8 位网文大神的创作经验特化而来

## 特性

- 激进的上下文管理策略
- 用项目管理的方式管理小说
- 特化工具和题材拆分
- 支持按需自动加载
- 创作自检引擎
- 使用桥接情节平滑过渡
- 由读者欲望驱动
- 支持三幕、五幕和六幕体系
- 支持因果链和支线扩展

## 使用方法

直接和 AI 说要用 `novel-creation-skill` 创作小说就可以啦

不过这里还是列出几点需要注意的

### 准备资料

如果你是同人文的话需要预先准备资料

本 SKILL 是从我自身需求出发的, 而我正好是一名同人小说作者, 所以本 SKILL 对同人小说的支持是最好的

首先第一步, 你需要准备你的同人目标的资料, 如果是游戏同人, 请提供 Wiki 地址

这里推荐几个我自己在用的:

- 二次元游戏: [GameKee](https://www.gamekee.com)
- B系游戏: [BiliWiki](https://wiki.biligame.com/wiki/)
- 海外游戏: [Fandom](https://games.fandom.com/zh/wiki/Wikia%E9%81%8A%E6%88%B2wiki%E5%88%97%E8%A1%A8)
- 中文(稍)小众: [灰机Wiki](https://www.huijiwiki.com/wiki/%E9%A6%96%E9%A1%B5)

那如果你是历史同人? 这类非常特殊, 你要是敢不提供资料让 AI 闷头写信不信它直接篡改历史, 建议历史同人认准百度百科、维基百科和[世界历史百科](https://www.worldhistory.org/trans/zh/)

还有影视同人, 这个我涉猎就比较少了, 只知道二次元的可以用[萌娘百科](https://moegirl.icu/Mainpage)

最后就是小说同人了, 这类你准备小说原著即可

资料都就绪后连着这句命令发给 AI 👉 `请使用 novel-creation-skill 的原著资料提取工具提取下列资料`

### 设计基础设定

在这一步你需要用正常的语言告诉 AI 你期望的是一个怎样的故事

这里我借鉴了 Superpowers 的头脑风暴

你只需要和 AI 表达自己的想法即可, 有疑问 AI 会主动询问你的

> 重点说一下: 不同的设定代表不同的小说类型, 建议先看下[SKILL 支持的类型](./novel-creation-skill/type), 不在支持范围内的效果可能无法达到预期

### 设计人物卡

这一步理论上是自动的, 在你设计好基础设定后, AI 会自动为你创建初始的主角和配角人物卡

如果你对他们的名字不满意, 我这里专门为大家准备了`人物取名工具`, 你可以直接让 AI 使用此工具来给你的角色取名

个人建议不要在会话中手动去修改`.novel`中的文件, 有想要修改的请直接和 AI 交流修改
